<Leetcode Hot 100>: 1.两数之和

Created2025-03-06|Updated2025-03-07
|Post Views:

LeetcodeHot100哈希表（HashMap）题目其一

1.题目概述

给定一个整数数组 nums 和一个整数 target，我们需要在数组里找到两个整数，它们的和等于 target，并返回这两个整数在数组中的下标。

  • 唯一解：题目保证给定的数组和目标值只会有唯一的一组答案，也就是说一定能找到那两个数，而且只有一对符合要求。
  • 不能复用元素：两个数必须是数组里不同的元素，所以它们不能是同一个位置上的数字。
  • 返回下标顺序随意：因为只要找出符合要求的下标即可，返回 [i, j] 和 [j, i] 都算对。

示例：

  1. nums = [2, 7, 11, 15], target = 9
    • 答案是 [0, 1]，因为 nums[0] + nums[1] = 2 + 7 = 9
  2. nums = [3, 2, 4], target = 6
    • 答案是 [1, 2]，因为 nums[1] + nums[2] = 2 + 4 = 6
  3. nums = [3, 3], target = 6
    • 答案是 [0, 1]，因为 nums[0] + nums[1] = 3 + 3 = 6

 

2.解题思路

2.1 方法一：两层循环（暴力求解）

思路很简单，两层嵌套循环暴力遍历即可，外层循环下标记为 i 从 0 到 len(nums) - 1，内层循环下标记为 j 从 i+1 到 len(nums) - 1

每次判断 nums[i] + nums[j] == target 是否成立，如果成立，就返回 [i, j]。如果全部找完还没有满足的组合，就说明没有解（但题目已保证一定有解，所以一定能找到）

这个解法的时间复杂度显然是O(n2)

class Solution(object):
    def twoSum(self, nums, target):
        # 外层循环
        for i in range(len(nums)):
            # 内层循环
            for j in range(i + 1, len(nums)):
                # 如果找到了和为 target 的组合
                if nums[i] + nums[j] == target:
                    return [i, j]

2.2 方法二：使用HASHMAP快速查找

为了使得解法的时间复杂度减少，就需要引入哈希表来避免暴力的双层循环了。在Python里面，常用的字典（dict）其实就是一个HASHMAP。

思路其实也不难，简单来说就是当我们想要寻找a 和 b，使得 a + b = target，那么只要知道 a，就可以推断 b = target - a。

也就是说，每次遍历到一个数字，只需要查看一下之前有没有存过一个数 nums[i]，使其可以满足一个数 b = target - nums[i]。如果之前就记录过 b 在什么下标，那么我们就得到了答案。

class Solution(object):
    def twoSum(self, nums, target):
        # 创建一个空字典，用于存储“数值 -> 下标”的映射
        hashmap = {}

        # 循环遍历 nums 列表
        for i, num in enumerate(nums):
            # 计算另一个需要的数字
            complement = target - num

            # 如果 complement 在字典中，说明之前就已经出现过那个数
            # 直接返回存储在字典中的下标和当前下标
            if complement in hashmap:
                return [hashmap[complement], i]

            # 如果没有，则将当前数与其下标存入字典
            hashmap[num] = i

        # 按题意一定能在循环中找到答案，此处可以不写
        # 不过为了语法完整或在其他情况下也可以写 return []

以上就是解决这道题的思路。感谢观看！
