<Leetcode Hot 100> 4:移动零

Created2025-04-11|Updated2025-04-26|LeetcodeHot100
|Post Views:

LeetcodeHot100双指针（Two Pointers）题目其一

1. 题目概述

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

请注意 ，必须在不复制数组的情况下原地对数组进行操作。

示例 1:

输入: nums = [0,1,0,3,12]
输出: [1,3,12,0,0]

示例 2:

输入: nums = [0]
输出: [0]

提示:

  • 1 <= nums.length <= 104
  • -231 <= nums[i] <= 231 - 1

进阶：你能尽量减少完成的操作次数吗？

2. 解题思路

从第一看到题目的第一反应，可以很快想到两个暴力解法：

其一是遍历数组，那就是一遇到0就放到末尾去，但是显然按照这个思路推导下去的话：原来的顺序就会被打破了。题目里面明确要求了保持非0元素的相对顺序。

其二是再创建一个数组，如此我们可以轻易地解决这个问题：把所有非零的元素放入新数组，然后剩下的元素全部放置0，显然就达到目的。但是我们可以看到题目明确要求：必须再不复制数组的情况下原地对数组进行操作，所以这个思路中拓展了占用空间的想法显然也被直接否决。

如此，我们即可引出此题目的最佳解题思路：双指针（Two Pointers）方法。

所以这道题使用双指针的逻辑在哪里呢？双指针这个技术带来的优势我个人觉得可以抽象成：在一次遍历的过程中干两件事。我们上面想法之所以发生冲突的原因就是因为这道题的“移动0”的要求实际上由两个重要的PART构成：

  • 把0移动到后方

  • 保证原非0数的顺序不变

    所以我们只需要一快一慢的双指针，慢指针用来存放非零数据，快指针用来遍历所有元素显然目标就可以轻松达成了。双指针（Two Pointers）之于这道题目的思路划分成步骤：

  1. 使用 左指针 left 记录非零元素应该放置的位置。

  2. 使用 右指针 right 遍历整个数组。（两个指针都从0开始，但是分管不同的任务，一快一慢）

  3. 如果 nums[right] 是非零数，就 把它和 nums[left] 交换，然后 left 前进。

  4. 这样，所有非零元素都会按顺序移动到前面，剩下的自动填充 0。

3. 代码实现

def move_zeroes(nums):
    left = 0  # 左指针，指向非零元素应存放的位置

    # 右指针遍历整个数组
    for right in range(len(nums)):
        if nums[right] != 0:
            # 交换 nums[right] 和 nums[left]，把非零数放到前面
            nums[left], nums[right] = nums[right], nums[left]
            left += 1  # 移动左指针，确保非零元素顺序正确

# 示例测试
nums1 = [0,1,0,3,12]
move_zeroes(nums1)
print(nums1)  # 输出: [1,3,12,0,0]

nums2 = [0]
move_zeroes(nums2)
print(nums2)  # 输出: [0]

显然这个方法只遍历一次即可完成，时间复杂度为O(n),而并没有创建新的数组出来，所以空间复杂度是O(1)，可以说是相当优秀的解法了，泪目😭。

以上就是这道题目的全部内容了。非常感谢观看。
