<Leetcode Hot 100> 5.盛最多水的容器

Created2025-04-24|Updated2025-04-26|LeetcodeHot100
|Post Views:

LeetcodeHot100双指针（Two Pointers）题目其二

1. 题目概述

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

说明：你不能倾斜容器。

示例 1：

输入：[1,8,6,2,5,4,8,3,7]
输出：49
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

示例 2：

输入：height = [1,1]
输出：1

提示：

  • n == height.length
  • 2 <= n <= 105
  • 0 <= height[i] <= 104

2. 解题思路

构成承接雨水容器的容积显然由左右两壁（相对较矮者）以及相距距离来决定。

我们没有办法直接通过器壁高度来确定容积，因为不同组合的情况下容器宽度不同对容积的影响也很大。故而需要遍历来确定到底什么组合可以产生最大容积。

如果从暴力的角度来看的话，这个题目显然就是一个双重嵌套循环的解法。外层循环先“固定”下来一个器壁，然后循环选择另一个器壁，实时计算容积并维护一个最大值。但是这样做显然很低效，因为时间复杂度太高，中间选择了很多次明显不会承接更多水的组合（这个后面解释）乃至重复组合。

这道题很明显可以看出来用双指针的解法来解决。因为构成器壁的两个垂直线肯定是一个在左边找一个在右边找的，在此前提下我们直接使用两个指针，分别执行从前往后和从后往前的移动。那么具体是怎么个移动法呢？我们不可能每次两者都移动一个位置，那么就没什么意义，只是尝试了与中轴距离相等的所有组合，跟找到最优解并无关系。所以构建双指针法最重要的思想在于找到移动指针的约束条件。

回顾面积公式：

$$
面积=宽度×min(左边高度,右边高度)
$$

从中可以看出，高度由短板决定，显而易见的我们每次去移动短的那一端就可以。原因是每次移动后宽度必然会减小，所以说高度必须增加才有可能获得容积更大的方案。但是高度又由短板决定，所以移动短板，使其有可能由一个更高的板替换才是正解，这样才具有提高容积的可能性。

也许有人会问，那老子要是就是每次要移动长板又如何呢？难道就不可能拿到更高容积吗？当然不行。我们可以做如下推演：

  1. 老子对长板处指针进行了移动，接下来可能会：

    • 相较于当前短板，仍然换了一个更长的板子进来。

    • 相较于当前短板，换了一个新的更短板进来作为新的短板。

  3. 那么对老子们显而易见，前者宽度减少了但是高度不变，所以容积减小。后者宽度减少且高度减少，所以容积减少。那么这样的话绝对不可能拿到更大容积的容器。

所以这样推演的意义在哪呢？bro其实无意中已经推导出来了找最小容器的思路了。实际上如果真要找最小容器的话，左右指针的推移也可以从中间开始来确保宽度刚开始最小，然后继续去找短板也行，这样每次更新就从“可能有短板变长的机会”变成了“可能有短板变短的机会”来对抗“宽度每次移动都会增加”这个事实。

3. 代码实现

#include <vector>
#include <algorithm>

class Solution {
public:
    int maxArea(vector<int>& height) {
        int left = 0; //左指针
        int right = height.size()-1; //右指针
        int max_area = 0; //最大面积

        while(left<right){
            //计算当前的面积
            int width = right - left;
            int h = min(height[left],height[right]);

            int area = width*h;

            //更新最大面积
            max_area=max(area,max_area);

            //每次移动短的那一根指针
            if(height[left]<height[right]){
                left++;
            }else{
                right--;
            }

        }   
        return max_area;     
    }
};

解题思路以上。用到了c++的algorithm库里的min和max方法。
